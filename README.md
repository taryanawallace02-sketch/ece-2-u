# ECE 2 U

## Overview
ECE 2 U is a web application designed to support early childhood education by providing resources for both teachers and parents. The app features a lesson plan generator for teachers and a behavior support guide for parents, along with interactive tools to enhance learning and development at home and in the classroom.

## Features
- **Teacher Side:**
  - Lesson plan generator with age group and theme selection.
  - Behavior management scripts for common classroom situations.
  - Movement break section with music suggestions and a countdown timer.
  - Turn-taking timer with gentle chime sound.

- **Parent Side:**
  - Behavior support guide with scripts and tips for challenging behaviors.
  - Materials finder for generating activity ideas based on household items.
  - Home activities that mirror classroom learning.
  - Rotating curated articles on various parenting topics.
  - Calm down corner guide with step-by-step instructions.

## Technologies Used
- HTML
- CSS
- JavaScript

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the application.
3. Ensure you have a modern web browser for the best experience.

## Usage Guidelines
- Select your role (Teacher or Parent) on the homepage to access the respective features.
- Use the language selector at the top of each page to switch between supported languages.
- Follow the prompts to generate lesson plans, behavior scripts, or activity ideas.
